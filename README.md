## Quick Start

1. Make sure you have recent, [stable](https://nodejs.org/en/about/releases/) version of node (>= 12.16.1).

```
   nave use stable
   node -v
```

2. Clone or download this repo.

3. Get latest releases of the tools

```
   npm update --save
```

## Commands

### Lint

```
npm run lint
```

### Build

```
npm run build
```

### Test

#### Unit test

```
npm run test
```

#### Coverage

```
npm run test-cov
```
